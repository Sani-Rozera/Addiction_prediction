#  Social Media Addiction Prediction

A Machine Learning web app that predicts a user's **social media addiction score** based on lifestyle and usage patterns.

Built using Python, Scikit-learn, and Streamlit.


-> How to Run

pip install -r requirements.txt
streamlit run app.py

-> Model

* Algorithm: Linear Regression
* Trained using Scikit-learn
* Model saved as `Addiction.pkl`


-> Tech Stack

Python • NumPy • Pandas • Scikit-learn • Streamlit
